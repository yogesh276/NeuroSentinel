# NeuroSentinel AI

## 🛡️ Overview

NeuroSentinel AI is an intelligent cyber threat detection and response system inspired by neuromorphic computing principles. The platform monitors system behavior and identifies potential cyber threats in real time using adaptive threat analysis techniques.

The system evaluates suspicious activities such as failed login attempts, unusual data transfers, and unknown processes to determine the severity of potential attacks. Based on the detected threat level, NeuroSentinel AI recommends appropriate mitigation actions to enhance cybersecurity resilience.

## 🧠 How It Works

The system collects security-related indicators:

* Failed Login Attempts
* Data Transfer Activity
* Unknown Process Detection

A threat score is generated based on these parameters.

### Risk Levels

| Threat Score | Risk Level      |
| ------------ | --------------- |
| Low          | Safe            |
| Medium       | Warning         |
| High         | Critical Threat |

When a high-risk threat is detected, NeuroSentinel AI recommends:

* Block Suspicious IP Address
* Quarantine Affected Device
* Alert Security Administrator
